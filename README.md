# Silverstone Components

General purpose web-components built in StencilJS

## Installation

```
$ npm i @silverstone/components
```

Consult [the Stencil docs](https://stenciljs.com/docs/overview) for details on integrating into your framework of choice.

## Components

### Button

```html
<ss-button>Click me</ss-button>
```

### Modal

```html
<ss-modal>
  <h1 slot="title">My Modal</h1>
</ss-modal>
```
